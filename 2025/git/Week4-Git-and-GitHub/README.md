## Tools & Commands Used
| Category | Commands |
|-----------|-----------|
| Git Basics | `git init`, `git add`, `git commit`, `git clone`, `git push`, `git pull` |
| Branching | `git branch`, `git switch`, `git checkout`, `git merge` |
| Undoing | `git reset`, `git revert`, `git restore` |
| Stashing | `git stash`, `git stash pop`, `git stash apply` |
| Advanced | `git cherry-pick`, `git rebase`, `git log`, `git diff` |
| Authentication | `git remote`, `git config`, SSH, PAT tokens |
