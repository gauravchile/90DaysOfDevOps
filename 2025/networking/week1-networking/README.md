# Week 1 — Networking Fundamentals

This folder documents the foundational networking concepts learned during **Week 1** of the **90 Days of DevOps — 2025 Edition** challenge. Understanding how data moves across systems, how protocols function, and how to secure network communication is critical for every DevOps professional.

---

## Objectives

* Understand the **OSI and TCP/IP Models** and their layers.
* Learn about **common networking protocols and port numbers** used in DevOps workflows.
* Configure **AWS Security Groups** for cloud instance security.
* Practice **essential networking commands** for troubleshooting and system monitoring.

## Key Takeaways

* Networking is the backbone of DevOps, cloud, and containerized environments.
* Secure configuration of ports and protocols is essential to system reliability and safety.
* Practical understanding of Security Groups and network tools builds a strong foundation for cloud and CI/CD work.

## How to Use

1. Read through each `.md` file to understand core concepts.
2. Practice the commands and steps in your local environment or cloud lab.
3. Apply the learned concepts to real-world scenarios such as troubleshooting, setting up firewalls, or configuring cloud security.
