## Overview
This week focuses on essential Linux administration and automation topics.  
You will explore user management, permissions, log analysis, disk handling, process management, and shell scripting.  
These are foundational skills for any DevOps engineer managing production systems.

---

## Topics Covered
1. User and Group Management  
2. File Permissions  
3. Log Analysis  
4. Volume and Process Management  
5. Automation and Backups  

---

## Objectives
- Understand how Linux manages users, groups, and privileges.  
- Learn file permissions, ownership, and access control.  
- Analyze and manipulate logs using `grep`, `awk`, and `sed`.  
- Manage volumes, storage, and system processes effectively.  
- Automate routine tasks using shell scripting and cron jobs.  

---

## Tools and Commands Used

| Category | Tools |
|-----------|-------|
| User Management | `useradd`, `groupadd`, `usermod`, `passwd`, `visudo` |
| Permissions | `chmod`, `chown`, `umask`, `stat`, `ls -l` |
| Logs | `grep`, `awk`, `sed`, `sort`, `uniq` |
| Processes | `ps`, `top`, `htop`, `kill`, `df`, `du`, `mount` |
| Automation | `bash`, `tar`, `cron`, `echo`, `date` |

---

## Learning Outcomes
By completing this week, you will be able to:
- Manage Linux users and groups confidently.  
- Apply secure file and directory permissions.  
- Perform log parsing and sanitization for insights.  
- Monitor system resources and processes efficiently.  
- Build and schedule simple automation scripts.
